# C语言题目后台助手

后台常驻桌面小工具，全局快捷键触发，托盘颜色直观反馈状态，AI 辅助生成 C 语言答案。

## 部署

```powershell
# 1. 克隆
git clone https://github.com/HermosMerlin/FuckMati.git
cd FuckMati

# 2. 创建虚拟环境并安装依赖
python -m venv .venv
.venv\Scripts\activate
pip install -e ".[dev]"

# 3. 第一次运行会自动生成 config.json，编辑填入你的 api_key
notepad config.json
```

## 使用

```powershell
# 前台运行（看日志）
.\run.bat

# 后台静默运行
.\start-hidden.vbs
```

### 状态机与快捷键

托盘图标右下角的彩色圆点实时显示系统状态：

| 颜色 | 状态 | 含义 |
|------|------|------|
| 🟢 绿色 | **READY** | 系统就绪，可以按快捷键触发 |
| 🟡 黄色 | **REQUESTING** | 正在读取剪贴板并向 AI 请求答案 |
| 🔵 蓝色 | **TYPING** | 正在模拟键盘逐字输出答案 |
| 🔴 红色 | **ERROR** | API 连接失败，10 秒后自动重试 |
| ⚫ 灰色 | **IDLE** | 启动中或重试中，正在检测 API |

**按一次 `Ctrl+Alt+G`，系统会自动完成一个完整阶段，不需要反复按键。**

### 典型操作流程

1. **把题目复制到剪贴板**（Ctrl+C）
2. **按一次 `Ctrl+Alt+G`** → 状态变为 🟡 黄色（请求中）
   - 系统自动：读取剪贴板 → 发送给 AI → 等待返回 → 缓存答案
   - 完成后自动回到 🟢 绿色（就绪）
3. **等圆点变回绿色后，再按一次 `Ctrl+Alt+G`** → 状态变为 🔵 蓝色（输出中）
   - 系统自动：将 AI 答案以打字机效果逐字输入到当前光标位置
   - 完成后自动回到 🟢 绿色（就绪）

> 💡 **为什么需要按两次？** 第一次是"问 AI"，第二次是"把答案打出来"。你也可以在步骤 2 之后先去别的地方，过一会儿再回来按第二次快捷键输出答案——答案会缓存在本地，不会丢失。

### 注意事项

- **黄色/蓝色状态下按快捷键会被忽略**：防止在请求或输出过程中误触导致中断。
- **红色状态下按快捷键无效**：系统正在自动重试连接 API，10 秒后会自动恢复。
- **强制重置**：如果卡住了，右键托盘图标选择"强制重置为空闲"，系统会立即回到灰色重新检测 API。
- **托盘右键可退出**。
